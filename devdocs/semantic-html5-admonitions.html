One admonition markup, five resolved styles

The point of the semantic converter is that styling stays easy: every section below styles the exact same HTML — the raw output of the semantic-html5 backend — with a few scoped CSS rules. No wrapper <div>s, no layout <table>, no markup changes; even the icons are added in pure CSS (a masked pseudo-element on .label).

<aside class="admonition note" role="note">
<strong class="label">Note</strong>
