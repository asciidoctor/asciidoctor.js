This document exercises most of the AsciiDoc syntax so the default stylesheet of the `semantic-html5` backend can be evaluated at a glance.
It is the preamble: it precedes the first section and contains a footnote.footnote:[Footnotes collect into a semantic list at the end of the document.]

== Text formatting

Regular paragraph text with *bold*, _italic_, `monospace`, *_bold italic_*, `*bold mono*`, #highlighted#, ^super^script, ~sub~script, "double curved quotes", 'single curved quotes', and an en&#8211;dash&#8201;&#8212;&#8201;style replacement.

Styled spans: [.underline]#underlined#, [.line-through]#struck through#, and [.nowrap]#a span that refuses to wrap#.

A sentence ending with a hard line break +
lands on the next line.

== Admonitions

NOTE: Useful information that users should know, even when skimming content.

TIP: Helpful advice for doing things better or more easily.

IMPORTANT: Key information users need to know to achieve their goal.

WARNING: Urgent info that needs immediate user attention to avoid problems.

CAUTION: Advises about risks or negative outcomes of certain actions.
