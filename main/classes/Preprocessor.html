    Class Preprocessor

    Preprocessors are run after the source text is split into lines and normalised, but before parsing begins.

    Asciidoctor passes the document and the document's Reader to the process method of the Preprocessor instance. The Preprocessor can modify the Reader as necessary and either return the same Reader (or falsy) or a substitute Reader.

    Implementations must extend Preprocessor.
